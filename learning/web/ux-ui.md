### difference between UI design and UX design

**UI**: \
컴퓨터, 웹사이트, 시스템등의 정보기기와 user가 만나는 화면\
정보기기와 user 상호작용 매개체

**UX:**\
정보기기를 사용할때 user가 직, 간접적으로 경험하는 모든 요소
