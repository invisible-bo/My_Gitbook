# Responsive web design(RWD)

* 다양한 디바이스(PC, 태블릿, 스마트폰 등)에서 웹사이트가 최적화되도록 화면 크기를 자동으로 조절하는 웹 개발 개념&#x20;

### Viewport

* 화면에 보여지고 있는 영역. 기기별로 다르다
* HTML로 개발시 meta tag로 viewport설정
