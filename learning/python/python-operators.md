# Python Operators

1. 산술 연산자 (Arithmetic Operator)

더하기

```python
print(3 + 2) → 5
```

빼기

```python
print(3 - 2) → 1
```

곱하기

```python
print(3 * 2) → 6
```

실수(Floating-Point)나눗셈

```python
print(3/2) → 1.5
```

정수나눗셈

(결과는소수점 이하를 버린 정수)

```python
print(3//2) → 1
```

나머지

```python
print(3%2) → 1
```

제곱

```python
print(3**2) → 9
```



2. 비교 연산자 (Comparison Operators)

같음, 다름, 크다, 작다, 크거나 같다, 작거나 같다

```python
print(3==3) → True
print(3!=2) → True
print(3>2) → True
print(3<2) → False
print(3>=2) → True
print(3<=2) → False
```



3. 논리 연산자 (Logical Operators)

and

```python
True and False → False
```

or

```python
True or False → True
```

not&#x20;

```python
not True → False
```



4. 할당 연산자 (Assignment Operators)

값 할당&#x20;

```python
(x = 5)
```

더해서 할당

```python
(x += 3) → x = 8
```

빼서 할당

```python
(x -= 2) → x = 6
```

곱해서 할당

```python
(x *= 3) → x = 18
```

나눠서 할당

```python
(x /= 3) → x = 6
```

나머지 할당

```python
(x %= 2) → x = 0
```

몫 할당

```python
(x//= 2)
```

제곱 할당

```python
(x **= 2)
```



5. 비트 연산자 (Bitwise Operators)

비트 AND

```python
(5 & 3) → 1
```

비트 XOR

```python
(5 ^ 3) → 6
```

비트NOT

```python
~5 → 6
```

왼쪽 shift

```python
5 << 1 → 10
```

오른쪽 shift

```python
5 >> 1 → 2
```
